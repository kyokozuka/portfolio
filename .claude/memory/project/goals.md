# 目標とマイルストーン

## ビジョン

ポートフォリオは「クラウドスケールのインフラエンジニアリング」と「人間中心の UX デザイン」の両領域を示す。Signal/Noise デザイン言語により、シンプルかつ専門性の高い印象を与える。
