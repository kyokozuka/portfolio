# 制約と軽微な決定ログ

## 設計制約

### ビジュアル制約
- **単一のアクセントカラー**
  - テラコッタ色（`#C8552A`）のみ；複数色は許可されない
  - Signal/Noise の厳密さを保つ；色追加は新 ADR が必要

- **タイポグラフィは 2 書体のみ**
  - Cormorant Garamond（見出し）+ DM Sans（本文）
  - システムフォールバック使用（日本語対応）

- **正方形のコーナー（border-radius なし）**
  - 建築的、グリッドベースの美学
  - `rounded-none` または `rounded` なし

### インタラクション制約
- **アニメーションは最小化**
  - 待機中のページは静止；操作時のみ動く
  - 継続的なループアニメーション禁止（floating particles など）

- **ホバー効果は色・ボーダー遷移のみ**
  - `translate-y` や `scale` は使用禁止
  - 遷移期間は 200–400ms 程度

## スコープ決定

### 含まれるもの
- 4つの主要ページ（home, software, uiux, contact）
- Signal/Noise デザインシステムの完全実装
- セマンティック HTML + アクセシビリティ対応
- 二言語対応（EN / JP）

### 明確に含まれないもの
- ブログ機能（今後の拡張）
- CMS・バックエンド（静的コンテンツのみ）
- 実装デモ・インタラクティブプロジェクト詳細（リンク参照）
- ダークモード（Light ベース固定）
